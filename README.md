## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/4429BrandingnMarketing/4429BrandingnMarketing.git
cd 4429BrandingnMarketing
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables** (optional for now)
```bash
cp .env.example .env
# Edit .env with your API keys when ready
```

4. **Start development server**
```bash
npm run dev
```

5. **Open in browser**
```
http://localhost:3000
```

## ğŸ“¦ Deployment

### Deploy to Vercel (Recommended)

1. **Install Vercel CLI**
```bash
npm install -g vercel
```

2. **Deploy**
```bash
vercel
```

3. **Follow the prompts** - Vercel will automatically detect Vite and configure everything

### Deploy to Netlify

1. **Build the project**
```bash
npm run build
```

2. **Deploy the `dist` folder** to Netlify via:
   - Drag & drop on netlify.com
   - Netlify CLI
   - Connect your GitHub repo

### Deploy to other platforms

The app builds to static files in the `dist` folder. You can deploy these anywhere:
- GitHub Pages
- AWS S3 + CloudFront
- Digital Ocean
- Railway
- Render
